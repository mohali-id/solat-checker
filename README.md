# Solat Checker ✨

Aplikasi web sederhana untuk tracking solat 5 waktu harian.

## Fitur

✅ List 5 waktu solat (Subuh, Dzuhur, Ashar, Maghrib, Isya)  
✅ Klik untuk tandai sudah solat (dengan timestamp)  
✅ Status disimpan di session  
✅ Auto reset setiap ganti hari  
✅ Tampilan responsif & modern  

## Cara Menjalankan

```bash
npm start
```

Buka browser: http://localhost:3000

## Teknologi

- Node.js + Express
- Express Session (untuk menyimpan status)
- Vanilla JavaScript (frontend)
- Auto-reset harian berdasarkan tanggal
